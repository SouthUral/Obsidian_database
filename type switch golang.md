[[switch golang]]

Инструкция ==switch== выбора по типу в языке Golang имеет следующий синтаксис:
```go
switch необязательнаяИнструкция; переключательТипа {
case списокТипов1:
	блок1
case списокТиповN:
	блокN
default:
	блокD
}
```
Переключатель типа - это выражение, значением которого является тип.
